# Generator-koa-cli

> Is a cli for creating koa project projects work.

创建支持`typescript`和`es`开发的基于koa模板。

## Features

- support typescript.

## Requirements

```bash
node >= 7.6
yarn >= 0.22
npm >= 4.x
```

## Use

```bash
# Install
$ npm i generator-koa-cli -g

# Create app
$ far ts myapp

# Start app
$ cd myapp
$ npm i
$ npm start
```
